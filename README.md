# The Groom Room Website

## Project Description
Full source code written in HTML/CSS/JavaScript for the Groom Room website that is a small locally owned dog grooming business. The main page, titled index.html, brings up the front page
of the website that gives a brief description of the business and links to the other pages on the website. 

## Install
Just clone the repository and click the index.html file, will work on any modern internet browser.
